# 🏠 NestFind

Ethiopia's premier house rental management system.

## Stack

| Layer     | Tech                                          |
| --------- | --------------------------------------------- |
| Frontend  | React 18, Vite, React Router v6, Tailwind CSS |
| Backend   | Node.js 18, Express.js                        |
| Database  | MongoDB, Mongoose                             |
| Auth      | JWT + bcrypt + OTP                            |
| Real-time | Socket.io                                     |
| Files     | Cloudinary                                    |
| Email     | Nodemailer                                    |
| SMS       | Twilio                                        |
