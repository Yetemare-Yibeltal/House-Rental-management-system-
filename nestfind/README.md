# 🏠 NestFind

Ethiopia's premier house rental management system.  
Connect tenants with verified landlords — browse, book, sign, pay, all in one place.

---

## Tech Stack

| Layer        | Technology                                    |
| ------------ | --------------------------------------------- |
| Frontend     | React 18, Vite, React Router v6, Tailwind CSS |
| Backend      | Node.js 18, Express.js, REST API              |
| Database     | MongoDB, Mongoose ODM                         |
| Auth         | JWT (access + refresh), bcrypt, OTP           |
| Real-time    | Socket.io                                     |
| File Storage | Cloudinary                                    |
| Email        | Nodemailer (Gmail SMTP)                       |
| SMS          | Twilio                                        |
| Deployment   | Vercel (client), Railway (server)             |
